Multi-Site Network Project

×

This Cisco Packet Tracer project demonstrates a multi-site network topology consisting of three interconnected sites. Each site is managed through routers and multilayer switches to ensure seamless communication across different subnets. The topology includes DNS, DHCP, and HTTP servers that provide essential services like automatic IP allocation, domain resolution, and web hosting to connected devices, ensuring a realistic enterprise-level simulation.

The project highlights concepts such as inter-VLAN routing, server configuration, and WAN connectivity between sites. By simulating real-world scenarios like centralized services and distributed client systems, the design provides a strong foundation in networking practices. This setup showcases how routers, switches, and servers interact to form a scalable and efficient communication network suitable for business or institutional use.

Centralized Network Model Project

×

The Domain Controller project is designed to simulate an enterprise-level IT infrastructure using Windows Server. By configuring a centralized domain, the system provides authentication, authorization, and secure access management for all connected client machines. Active Directory services were implemented to manage users, groups, and organizational units efficiently, ensuring structured and secure resource allocation.

This setup highlights the importance of centralized network administration within corporate environments. Features like Group Policy, file sharing, and user account management were integrated to demonstrate real-world scenarios of organizational IT operations. The project showcases how domain controllers streamline security and resource management, making it an essential solution for businesses that require scalable and reliable network systems.

Green Cart

×

The Green Cart project is a database-driven application designed to manage core operations of an online grocery system. Built on MySQL, it includes well-structured tables for products, categories, suppliers, customers, employees, sales, and inventory. The system ensures data integrity through relational database concepts and enforces constraints to maintain accuracy in product availability, customer details, and transactional records.

This project highlights the practical implementation of database management principles including normalization, primary/foreign keys, joins, and queries for CRUD operations. It demonstrates how relational databases can power real-world e-commerce platforms by enabling efficient inventory tracking, supplier management, and customer transactions. Green Cart stands as a strong example of applying SQL concepts to solve everyday business challenges.
